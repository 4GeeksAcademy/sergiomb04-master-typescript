## Instalación en un clic

Abre en [Codespaces](https://codespaces.new/?repo=4GeeksAcademy/master-typescript-programming-exercises) o [Gitpod](https://gitpod.io#https://github.com/4GeeksAcademy/master-typescript-programming-exercises).

## Instalación local

1. Instala LearnPack:

```bash
npm i @learnpack/learnpack -g
```

2. Instala el plugin local de TypeScript para LearnPack:

```bash
learnpack plugins:install file:$(pwd)/learnpack-typescript-plugin
```

3. Inicia los ejercicios:

```bash
learnpack start
```

## Bloques curriculares

### Skill 7.2: Aprender a nombrar o etiquetar flujos con funciones

- **Rango:** `001-042`
- **Enfoque:** declaración de funciones, parámetros, tipo de retorno, procedimientos `void`, datos primitivos.
- **Dificultad:** principiante a intermedio-bajo.

### Skill 8.1: Colecciones (arreglos y matrices)

- **Rango:** `043-115`
- **Enfoque:** recorrido de arreglos, transformaciones, filtros, agregaciones, acceso o transformación de matrices y búsqueda binaria.
- **Dificultad:** principiante a intermedio.

### Skill 8.3: Datos estructurados de tipo objeto en TypeScript

- **Rango:** `116-170`
- **Enfoque:** interfaces, objetos literales vs interfaces, propiedades opcionales, datos anidados, mutación vs inmutabilidad y paso por valor o referencia.
- **Dificultad:** intermedio.

## Contrato de archivos por ejercicio

Cada ejercicio contiene:

1. `app.ts`: archivo principal oficial del estudiante (solo base inicial).
2. `app.js`: entry runtime para el botón `Run` de LearnPack.
3. `README.md` + `README.es.md`: instrucciones.
4. `test.js`: script de evaluación automática.

`app.ts` y `app.js` se mantienen intencionalmente como plantillas iniciales (sin lógica resuelta).
